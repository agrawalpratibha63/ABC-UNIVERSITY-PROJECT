# 🎓 ABC University ERP & Management Portal

A comprehensive, full-stack University Enterprise Resource Planning (ERP) system designed to streamline academic, administrative, and student-faculty interactions. Built with a robust Python/Flask backend and a modern, glassmorphism-inspired UI.

## ✨ Key Features

🔐 **Role-Based Access Control (RBAC)**
Secure, unified login system that smartly redirects users to their specific dashboards based on their roles:
*   **Admin Command Center:** Complete oversight of the university, including student directory, faculty roster, and fee management.
*   **Student Portal:** View enrolled courses, real-time grades, and academic standing.
*   **Faculty Dashboard:** Manage teaching assignments, view class rosters, and track committee memberships.

📊 **Live Data Analytics & Visualization**
*   Integrated **Chart.js** for real-time graphical representation of student enrollments and faculty distributions.
*   Dynamic **Attendance Heatmaps** generated natively to track campus engagement.

🤖 **AI-Powered Assistance**
*   Built-in smart Chatbot integrated directly into the portal to answer queries regarding student data, toppers, and faculty locations in real-time.

🔔 **Dynamic Local-Storage Notifications**
*   System-wide toast alerts and a notification bell that stores activity history (e.g., "New Student Registered") dynamically using browser local storage.

📄 **Academic Operations**
*   One-click simulated secure PDF generation for official student transcripts.
*   Complex MySQL queries seamlessly fetched and displayed via accordion UI for administrative reporting.

## 🛠️ Technology Stack

*   **Frontend:** HTML5, CSS3 (Glassmorphism & Dark/Light Mode toggle), JavaScript (Vanilla), Bootstrap 5, Chart.js, FontAwesome.
*   **Backend:** Python, Flask (Jinja2 Templating, Sessions, Flash messaging).
*   **Database:** MySQL (Relational Database Management, Complex JOINs, Aggregations).
